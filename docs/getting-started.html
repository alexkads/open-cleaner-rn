‚ö° Quick Start

Get up and running with Clean RN Dev in less than 2 minutes:

1

Launch the App

Open Clean RN Dev from your Applications folder, Start Menu, or application launcher.

2

Run Quick Scan

Click the "Quick Scan" button to automatically detect all cache files in your system.

3

Review Results

Check the scan results to see what cache files were found and how much space they're using.

4

Clean Selected Items

Click "Clean Selected" to remove the cache files and free up disk space.

üõ†Ô∏è Troubleshooting

App won't start on macOS

Solution: This is usually due to Gatekeeper security. Follow these steps:

  1. Go to System Preferences > Security & Privacy
  2. Click "Open Anyway" next to the Clean RN Dev warning
  3. Or run: sudo xattr -rd com.apple.quarantine /Applications/Clean\ RN\ Dev.app

Permission denied errors

Solution: Grant necessary permissions:

  • macOS: System Preferences > Security & Privacy > Full Disk Access
  • Linux: Run with sudo if needed, or check file permissions
  • Windows: Run as Administrator for system-level cleaning

Scan finds no files

Solution: This might happen if:

  • Your development environment is already clean
  • Cache directories are in non-standard locations
  • You haven't done React Native development recently

Try running a React Native project first to generate cache files.
